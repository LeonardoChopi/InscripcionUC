Cómo generar tu propio array con IA

Podés usar una IA con capacidad de análisis de imágenes (como Claude o ChatGPT-4o) para convertir automáticamente una imagen de tu cronograma en el formato que necesita este sistema. Seguí los pasos a continuación.

Pasos
1
Tomá una captura de pantalla o foto de tu cronograma de clases (la tabla con materias, grupos, días y horarios).
2
Abrí tu IA preferida: Claude, ChatGPT o Gemini (Recomendada). Asegurate de usar una versión con soporte de imágenes.
3
Copiá el prompt de abajo, adjuntá tu imagen y enviá el mensaje.
4
Copiá el array que te devuelve la IA y úsalo en la página Cargar nuevo array.
Prompt a usar
📋 Prompt — copiar y pegar en la IA
Extrae datos de un cronograma de clases y conviértelos en un array JS llamado materias. Estructura de cada objeto: { id, nombre, tipo, grupo, salon, dia, inicio, fin } Reglas: - Un objeto por cada día con horario. - id autoincremental. - nombre: asignatura de la tabla. - tipo: "Teorico" o "Practico". - grupo: nombre del grupo. - salon: "AULA - PISO" o "VIRTUAL". - dia: Lunes | Martes | Miercoles | Jueves | Viernes. - inicio y fin en formato HH:MM. Devuelve SOLO: const materias = [ ... ]; Imagen a procesar: [ADJUNTAR IMAGEN DE LOS HORARIOS ]
