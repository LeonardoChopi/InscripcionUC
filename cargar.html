Campos requeridos por objeto
idNúmero único autoincremental
nombreNombre de la asignatura
tipo“Teorico” o “Practico”
grupo“Grupo 1”, “Grupo A”, etc.
salon“305 - 3er Piso” o “Virtual”
diaLunes | Martes | Miercoles | Jueves | Viernes
inicioHora en formato “HH:MM”
finHora en formato “HH:MM”
Ejemplo mínimo
const materias = [ { id: 1, nombre: "Fisica 1", tipo: "Practico", grupo: "Grupo 1", salon: "305 - 3er Piso", dia: "Miercoles", inicio: "10:00", fin: "11:30" }, ];
Notas
Un grupo con clases en múltiples días necesita un objeto por cada día. El sistema los agrupa automáticamente en una fila por grupo.

Podés generar el array con IA usando el prompt en la página principal, pestaña “Cómo generar mi array”.
ⓘ Cómo usar el formulario
1. Completá los datos de una clase.
2. Hacé clic en “Agregar”.
3. Si un grupo tiene clases en varios días, agregá una entrada por cada día con el mismo nombre y grupo.
4. Cuando termines, clic en “Guardar y aplicar”.
Grupos con varios días
Si Física 1 Grupo 1 tiene clases el Miércoles y el Viernes, agregás dos entradas con el mismo nombre y grupo, cambiando solo el día y horario. El sistema las une en una sola fila.
Salón: formato sugerido
305 - 3er Piso
Virtual 01
Actos - 1er Piso
ⓘ Sobre el almacenamiento
Los datos se guardan en localStorage de este navegador. No se envían a ningún servidor.

No ingreses datos sensibles como contraseñas o información personal.
